Moved Files:

Eclipse projects. Here's how the projects were restructured:

In ProductDemo-CordovaLib:
* src/ => productDemoCordovaLib/src/main/java/
In ProductDemo:
* AndroidManifest.xml => productDemo/src/main/AndroidManifest.xml
* assets/ => productDemo/src/main/assets/
* res/ => productDemo/src/main/res/
* src/ => productDemo/src/main/java/
